## 🛠️ Overview

ManasPDF is built to render PDF files quickly and efficiently. This application provides a simple interface for viewing PDF documents. It's designed to take full advantage of GPU capabilities to enhance performance.

ManasPDF is optimized for modern Windows systems. Recommended requirements include:

- RAM: 4 GB minimum
- GPU: Any Direct2D compatible graphics card
